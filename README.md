# backroom 0.0.1
three.js experimental backroom<br>
[launch demo](https://lo-th.github.io/backroom/)<br><br>

### Control

XSAD or ZSQD to move<br>
C to crouch<br>
SPACE to jump<br>

### About

No IA, source is hide for the moment.<br>
Generator use pieces of D&D dungeon v1.<br>
Render use three.js webgpu with ssgi.<br><br>
